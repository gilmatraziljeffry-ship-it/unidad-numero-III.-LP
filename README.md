#### unidad-numero-III.-LP


Este proyecto consiste en el desarrollo de un Sistema de Gestión de Productos en C#, diseñado bajo el paradigma de Programación Orientada a Objetos para administrar inventarios de forma segura y eficiente.
## Sus pilares fundamentales son:
Encapsulación Robusta: Protege la integridad de los datos mediante atributos privados y validaciones estrictas que impiden valores imposibles, como precios o cantidades de stock negativas.

Gestión de Colecciones: Implementa una clase Almacen que utiliza arreglos para almacenar múltiples productos, permitiendo realizar operaciones de registro, búsqueda y listado.

Algoritmos de Control: Incluye lógica para evitar códigos duplicados y un sistema de eliminación dinámica que reorganiza el arreglo automáticamente al borrar un ítem.

Interfaz Funcional: Ofrece un menú interactivo por consola que valida las entradas del usuario, garantizando una experiencia de uso fluida y libre de errores de ejecución.
